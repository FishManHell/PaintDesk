# Paint Desk

Paint Desk is a web-based drawing application built for creating and editing simple shapes directly on a canvas.  
It provides an interactive workspace where users can draw, move, zoom, and manage elements in a clean and minimal interface.

---

## Overview

The project was created as a browser-based paint tool focused on basic drawing functionality and canvas interaction.  
It allows users to work with multiple shape types, control stroke width and color, and manage canvas actions such as undo, redo, and clear.

---

## Features

- Freehand brush drawing
- Draw lines, squares, circles, and triangles
- Move elements on the canvas
- Zoom and navigate the workspace
- Undo and redo actions
- Clear the canvas
- Change stroke color
- Adjust stroke width
- Interactive canvas-based UI

---

## Main Functionality

- **Brush** — freehand drawing on canvas
- **Line** — create straight lines
- **Square** — draw square shapes
- **Circle** — draw circles
- **Triangle** — draw triangles
- **Move** — reposition elements
- **Zoom** — scale the workspace
- **Undo / Redo** — manage drawing history
- **Clear** — reset the canvas
